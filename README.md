# 🛁 Zift Unique - Trendyol Asistanı

Trendyol müşteri sorularına otomatik cevap üreten AI asistan.

## 🚀 Vercel'e Deploy (5 Dakika)

### Adım 1: GitHub'a Yükle
1. [GitHub.com](https://github.com)'da yeni repo oluştur: `zift-asistan`
2. Bu klasörün içindekileri repo'ya yükle

### Adım 2: Vercel'e Bağla
1. [Vercel.com](https://vercel.com)'a git, GitHub ile giriş yap
2. "Add New" → "Project" tıkla
3. `zift-asistan` repo'sunu seç
4. "Deploy" tıkla, bekle (2-3 dakika)
5. Bitti! `zift-asistan.vercel.app` gibi bir adres alacaksın

### Adım 3: API Anahtarı Al
1. [console.anthropic.com](https://console.anthropic.com) adresine git
2. Settings → API Keys → "Create Key"
3. Anahtarı kopyala
4. Uygulamada ⚙️ Ayarlar'a tıkla, anahtarı yapıştır

## 💰 Maliyet
- Vercel: Ücretsiz
- Claude API: ~$0.003 / soru (günde 100 soru = ayda ~$9)

## 🔧 Lokal Geliştirme
